=== SSS: SearchSomeSite

SSSはDLsiteの作品ページから直接BoothやFanzaでタイトル検索ページにタイトルを入れて検索してくれるやつです。


NOTE: 下記のコードは新しいタブのページで開きます。現在のタブで開きたい場合は *window.open(href, '_blank')* を *window.open(href, '_self')* に書き直してください。

----
include::js/SearchBooth.js[]
----
