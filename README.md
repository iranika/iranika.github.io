## おまけ：どうしてこのページは味気ないデザインなの？

古（いにしえ）のインターネットとホームページが大好きだからです。  
ほとんどテキストで書かれ、画像がチョットだけある。  
まさしく「文書」だった頃のホームページは今では絶滅危惧種になってしまいました。  
とは言え、現代のWebデザインは人類が試行錯誤を経て手に入れた優秀な道具です。
見やすく洗練されたデザインのWebページに出会ったら、先人たちに思いを馳せてみてください。
